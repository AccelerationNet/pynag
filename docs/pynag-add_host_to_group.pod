=head1 NAME

pynag-add_host_to_group - Adds a host to the specified nagios hostgoup

=head1 SYNOPSIS

pynag-add_host_to_group <hostname> <hostgroup_name>

=head1 DESCRIPTION

pynag-add_host_to_group adds the specified hostgroup to the host in question.

=head1 hostname

Hostname can be any correctly defined host in the nagios configuration.

=head1 hostgroup_name

hostgroup_name should be a valid hostgroup in the nagios configuration.
